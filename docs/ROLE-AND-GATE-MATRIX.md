# Role and Gate Matrix

| Role | Primary output | Exit gate |
|---|---|---|
| Solution Architect / Safety Owner | configuration and boundaries | source repos read-only; QC4-only; no cloud key dependency |
| Repository Contract Analyst | code/test evidence inventory | method/path/auth/payload clues retain file and line provenance |
| Postman/HAR Reconciliation Engineer | canonical endpoint model | duplicates merged; unmatched and conflicts retained |
| Collection Builder | Postman v2.1 and environment | importable structure; variables parameterized; no secret values |
| Manual Test Enablement Writer | mapping and guides | prerequisites, use, meaning, and limitations are readable |
| Validation Engineer | gate evidence | tests, secret scan, count check, SDK validation, repo status pass |
| Independent Reviewer | final summary | original mission reconciled to exact artifacts and counts |
